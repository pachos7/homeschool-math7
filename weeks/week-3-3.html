💡 Key Concepts

Using Multiplication and Division:

  • To solve equations with multiplication or division, use the inverse operation
  • If a variable is multiplied by a number, divide both sides by that number
  • If a variable is divided by a number, multiply both sides by that number
  • Always maintain balance: what you do to one side, do to the other

Examples:

  • If 3x = 15, divide both sides by 3 → x = 5
  • If x/4 = 7, multiply both sides by 4 → x = 28
  • If 2x = 18, divide both sides by 2 → x = 9

🌍 Real-World Connection

Multiplication and division equations help you:

  • Calculate unit prices and rates (e.g., price per item, miles per gallon)
  • Solve scaling problems (recipes, maps, construction)
  • Determine quantities when you know totals or averages
  • Work with ratios and proportions in real situations
