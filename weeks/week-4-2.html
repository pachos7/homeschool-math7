Week 4.2: Graphing DataSets

← Back to Part IV: Mathematical Change and Function Concepts

Objective

  • Understand how to graph the points in a coordinate plane using a table
  • Grasp the basic concept of the Y dependency on X

Tools

Desmos Graphing Calculator

Content

Exercise 1: Graph the points in the following table:

X Y

Additional Resources

Khan Academy: The Coordinate Plane

Exercise 2: Graph the points in the following table:

X 0 -1 -3 4 3 -4
Y 2 4 2 0 -3 -1

Exercise 3: Create a table of the points for a smiley face (use up to 12 points)

X
Y

Exercise 4: Create a table of the points in a horizontal line (use 5 points)

X
Y

Exercise 5: Create a table of the points in a vertical line (use 5 points)

X
Y

Challenges

Create a table of the points and see:

  • What happens if X and Y are always the same? For example: (-3, -3), (2, 2), (0, 0) etc.
  • What happens if Y is always double the value of X? For example: (-1, -2), (2, 4), (3, 6) etc.
  • What happens if Y is always 3 more than X? For example: (1, 4), (0, 3), (3, 6), (-4, -1) etc.
